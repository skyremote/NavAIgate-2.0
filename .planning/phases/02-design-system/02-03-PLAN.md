  <files>src/components/FloatingElements.tsx</files>
  <action>
Create FloatingElements for decorative background blobs:

Props:
- variant: 'hero' | 'section' | 'minimal' (predefined configurations)
- className: string (for positioning adjustments)

Each variant defines array of floating elements with:
- position (top/bottom/left/right percentages)
- size (w-20 h-20 to w-80 h-80)
- color (bg-blue-500/20, bg-purple-500/20, bg-teal-500/20)
- blur (blur-xl to blur-3xl)
- animation (float, float-reverse, pulse-slow)

Render as absolutely positioned divs with rounded-full.
Elements have pointer-events-none to not interfere with content.

Hero variant: 3 elements (top-left, top-right, bottom-left)
Section variant: 2 elements (top-left, bottom-right)
Minimal variant: 1 centered element
